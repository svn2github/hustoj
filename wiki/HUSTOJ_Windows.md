= Introduction =

Based on [http://www.colinux.org/ CoLinux], with Ubuntu12.04.


= Details =

Install:
  * Unzip downloaded file 
  * Click 启动hustoj.bat
  * Open http://127.0.0.1/
  * admin for OJ :
{{{
username:hustoj
password:freeproblemset
}}}
  * admin for OS :
{{{
username:root
password:root
}}}
  * To update code to latest svn version
{{{
sudo update-hustoj
}}}
